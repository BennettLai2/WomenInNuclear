This page will show you how to utilize this app to the fullest.

Admins

Creating an Admin Account

To create an admin account, the user must first create a normal account and an admin must give them admin priveliges.

To give admin privileges, log into the admin account and select "Make Admin" inside of the home page.

Delete Members Accounts

To delete a member's account, an admin must click on the "Destroy" button on the home page (shown in screenshot above).

Setting Up Milestones

To add a milestone, the admin must first select "View Milestones".

Then select "New Milestone". In this page, the current milestones can be seen, modified, and deleted as well.

After filling out the fields, the admin can click "Create Milestone"

Create/Modify/Delete Events

To create a new event, go to the "View Events" page and select "New Event".

This page also allows to see the current events, modify events, and delete events as well

To create a new event, fill out the fields. The meeting code will be autogenerated and shown right after creating.

Reset Points

To reset points, click on the "Reset Points" button on the homepage. This will reset all of the members' and admins' points back to 0

After clicking reset points, the admin must enter their email to confirm.

Users

Creating an Account

To create an account, click on the "Sign Up" button on the home page.

Fill out the corresponding fields and click "Sign Up".

Points

To see the number of current points, log into your user account.

View Event Schedule

To view the current events, click on "View Events" in the user's homepage.

View Leaderboard

To view the leaderboard of the current users with the highest points, click on "Leaderboard".

Attending a Meeting

To attend a meeting and get credit, add the Event ID in the provided field. Upon providing the correct Event ID (and assuming that the timing is still valid, this will lead to a new page with the event information.

After entering "Update User", the page will be redirected to the home page with the new points added.
